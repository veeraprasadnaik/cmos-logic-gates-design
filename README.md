 # CMOS Logic Gates Design using NMOS and PMOS

## 📌 About This Project

In this project, I have designed CMOS logic gates using NMOS and PMOS transistors. I have done simulations and observed how the circuits are working.

Mainly, I focused on understanding the behavior of CMOS circuits and how timing changes based on design.

## 🎯 What I Have Done

* Designed CMOS Inverter
* Designed NAND Gate
* Designed NOR Gate
* Used NMOS and PMOS transistors in all designs
* Performed simulations
* Observed timing analysis

## ⏱️ Timing Analysis

I have observed the following parameters:

* Propagation Delay
* Rise Time
* Fall Time

From the simulation results, I understood how output changes with respect to input and how delay occurs in circuits.

## 📊 My Observations

* CMOS circuits consume less power
* PMOS is slower compared to NMOS
* Rise time and fall time are different
* Delay depends on transistor size and load

## 🎓 Learning Outcomes

From this project, I learned:

* Basics of CMOS technology
* Working of NMOS and PMOS transistors
* How to design logic gates using CMOS
* How to perform timing analysis
* Understanding of propagation delay, rise time, and fall time
* Importance of transistor sizing in circuit performance
* How simulation tools help in analyzing circuits

## 🧪 Tools Used

* Microwind
* DSCH
* SPICE Simulation Tools

## 📈 Results

* Generated waveforms from simulation
* Calculated timing values
* Compared performance of different gates

## 📚 Conclusion

From this project, I learned how CMOS circuits work using NMOS and PMOS. I also understood timing behavior and how delay affects circuit performance.
